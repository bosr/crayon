get /runs/{runName}

All summaries (may be large) for run {runName}

  • Request
  • Response

URI Parameters

  • runName: required (string)

Query Parameters

  • plugins: (string)

    Filter output on these plugin names (comma separated list)

  • tags: (string)

    Filter output on these tag names (comma separated list)

  • format: (string - default: json)

    Format as JSON (or something else later)

  • last: (integer - default: -1)

    Return only the last n samples in each collection
